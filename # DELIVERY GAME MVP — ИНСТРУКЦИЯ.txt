# DELIVERY GAME MVP — ИНСТРУКЦИЯ ДЛЯ CLAUDE CODE

## ПРОЕКТ

Создать MVP мини-игры для доставки еды.

Название:
«Помоги курьеру довезти заказ»

Бренд:
АППЕТИТ

Тип проекта:
Маркетинговая мини-игра для вовлечения клиентов и выдачи промокодов/наград.

Главная цель:
Повышение удержания, вовлечения и конверсии в заказ.

---

# ТЕХНОЛОГИИ

Использовать:

* React
* Vite
* TypeScript
* HTML5 Canvas
* CSS modules или обычный CSS
* requestAnimationFrame

НЕ использовать:

* Tailwind
* Redux
* сложные библиотеки
* backend (в первой версии)

---
